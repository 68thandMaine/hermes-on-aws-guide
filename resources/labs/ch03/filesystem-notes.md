# Lab 3 — Filesystem exploration notes

Answer one sentence for each directory:

## `/`

The root directory. Everything in the filesystem is organized beneath `/`.

## `/home`

This is for the user home directories. Basically anything the developer creates
is put in here.

## `/etc`

This directory is for system configuration.

## `/var`

This directory is for logs, databases, spools, cache. Long running memory
basically.

## `/usr`

User programs and read only data.

## `/tmp`

Temporary files.
